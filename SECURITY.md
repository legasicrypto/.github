# Security Policy

## Supported Repositories

Security issues are accepted for active repositories in the `legasicrypto` organization.

## Reporting a Vulnerability

Do not open public issues for security findings.

- Email: `arthur@legasi.io`
- Include reproduction steps, impact, and affected repository/commit
- Include any proof-of-concept details needed to reproduce safely

We will acknowledge reports promptly, assess severity, and coordinate disclosure timing.
